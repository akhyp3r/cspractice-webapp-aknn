Official Edexcel Pseudocode Rules

These are the exact command styles used in the Pearson Edexcel International GCSE pseudocode command set.

Data types

INTEGER
REAL
BOOLEAN
CHARACTER
STRING

Use type coercion in input, for example (INTEGER) KEYBOARD.

Assignment

SET Counter TO 0
SET Sum TO Score + 10
SET Size TO LENGTH(Word)

Use SET ... TO. Do not use = for assignment in Edexcel pseudocode.

Input

RECEIVE Name FROM (STRING) KEYBOARD
RECEIVE Age FROM (INTEGER) KEYBOARD
RECEIVE YesNo FROM (CHARACTER) KEYBOARD

Use RECEIVE identifier FROM (type) device.

Output

SEND 'Have a good day.' TO DISPLAY
SEND Total TO DISPLAY

Use DISPLAY, not SCREEN.

Selection

IF Answer = 10 THEN
    SET Score TO Score + 1
END IF

IF Answer = 'correct' THEN
    SEND 'Well done' TO DISPLAY
ELSE
    SEND 'Try again' TO DISPLAY
END IF

Use END IF with a space.

WHILE loop

WHILE Flag = 0 DO
    SEND 'All well' TO DISPLAY
END WHILE

Pre-conditioned loop. It may run zero times.

REPEAT UNTIL loop

REPEAT
    SET Go TO Go + 1
UNTIL Go = 10

Post-conditioned loop. It runs at least once.

REPEAT n TIMES

REPEAT 100 - Number TIMES
    SEND '*' TO DISPLAY
END REPEAT

Count-controlled loop using an expression.

FOR loop

FOR Index FROM 1 TO 10 DO
    SEND ArrayNumbers[Index] TO DISPLAY
END FOR

FOR Index FROM 1 TO 500 STEP 25 DO
    SEND Index TO DISPLAY
END FOR

Use FROM, TO, optional STEP, and END FOR.

Arrays and strings

SET ArrayValues TO [1, 2, 3, 4, 5]
SET ArrayValues[0] TO 10
SET Sentence TO Sentence & Word & ' '

Indices start at 0. Use & to append/concatenate.

Subprograms

PROCEDURE CalculateAverage(Mark1, Mark2, Mark3)
BEGIN PROCEDURE
    SET Avg TO (Mark1 + Mark2 + Mark3) / 3
END PROCEDURE

FUNCTION AddMarks(Mark1, Mark2, Mark3)
BEGIN FUNCTION
    SET Total TO Mark1 + Mark2 + Mark3
    RETURN Total
END FUNCTION

Functions return a value. Procedures do not have to return a value.

Official Edexcel-Style Model Answers

1. Input 5 numbers and output the total

SET Total TO 0

FOR Count FROM 1 TO 5 DO
    RECEIVE Number FROM (INTEGER) KEYBOARD
    SET Total TO Total + Number
END FOR

SEND Total TO DISPLAY

2. Keep asking for a password until it is correct

REPEAT
    RECEIVE Password FROM (STRING) KEYBOARD
UNTIL Password = 'secret'

SEND 'Access granted' TO DISPLAY

3. Count how many array values are greater than 50

SET Scores TO [45, 67, 23, 89, 51]
SET Count TO 0

FOR Index FROM 0 TO LENGTH(Scores) - 1 DO
    IF Scores[Index] > 50 THEN
        SET Count TO Count + 1
    END IF
END FOR

SEND Count TO DISPLAY

4. Validate an age between 11 and 18

REPEAT
    RECEIVE Age FROM (INTEGER) KEYBOARD
UNTIL Age >= 11 AND Age <= 18

SEND 'Age accepted' TO DISPLAY

5. Find the largest number in an array

SET Numbers TO [4, 9, 2, 15, 6]
SET Largest TO Numbers[0]

FOR Index FROM 1 TO LENGTH(Numbers) - 1 DO
    IF Numbers[Index] > Largest THEN
        SET Largest TO Numbers[Index]
    END IF
END FOR

SEND Largest TO DISPLAY

6. Create a sentence using & append operator

SET Words TO ['The', 'Sky', 'is', 'grey']
SET Sentence TO ''

FOR EACH Word FROM Words DO
    SET Sentence TO Sentence & Word & ' '
END FOREACH

SEND Sentence TO DISPLAY

7. Use MOD to check if a number is even

RECEIVE Number FROM (INTEGER) KEYBOARD

IF Number MOD 2 = 0 THEN
    SEND 'Even' TO DISPLAY
ELSE
    SEND 'Odd' TO DISPLAY
END IF

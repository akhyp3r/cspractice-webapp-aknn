Logic Gate Rules

These are the rules you must know for Edexcel Computer Science logic questions.

NOT

Output is the opposite of the input.

0 → 1, 1 → 0

AND

Output is 1 only if both inputs are 1.

A AND B

OR

Output is 1 if at least one input is 1.

A OR B

NAND

NAND means NOT AND.

Output is 0 only when both inputs are 1.

NOR

NOR means NOT OR.

Output is 1 only when both inputs are 0.

XOR

XOR means exclusive OR.

Output is 1 when the inputs are different.

Exam-Style Logic Questions

These answers are written in the style a student should use in Paper 1.

1. Explain when an AND gate outputs 1.

An AND gate outputs 1 only when both inputs are 1. If either input is 0, the output is 0.

2. Explain when an OR gate outputs 1.

An OR gate outputs 1 when at least one input is 1. This means A can be 1, B can be 1, or both can be 1.

3. Explain the difference between OR and XOR.

An OR gate outputs 1 if one or both inputs are 1. An XOR gate outputs 1 only when the inputs are different, so it outputs 0 when both inputs are 1.

4. How many rows are needed in a truth table with 3 inputs?

A truth table with 3 inputs needs 8 rows because the number of rows is 2 to the power of the number of inputs. So 2³ = 8.

5. Describe A AND NOT B in words.

The output is 1 only when A is 1 and B is 0, because B is inverted before it is used in the AND gate.
